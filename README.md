# Davide D’Ignazio Portfolio

Portfolio personale sviluppato con Laravel, Vite, CSS custom e interfaccia responsive moderna.

## Obiettivo

Il progetto presenta il profilo professionale di Davide D’Ignazio come Full Stack Web Developer, con focus su Laravel, AI Integration, UX/UI e progetti web moderni.

## Tecnologie utilizzate

- Laravel
- Blade Components
- Vite
- JavaScript
- CSS Custom
- Responsive Design
- Git/GitHub

## Sezioni principali

- Hero section
- About
- Skills
- CV download
- Featured Project: VendoHub AI
- Case Study dedicata
- Contatti

## Featured Project

### VendoHub AI

Marketplace intelligente multi-categoria progettato con Laravel, Livewire, Bootstrap, Fortify e Google Cloud Vision AI.

La piattaforma permette agli utenti di pubblicare, gestire e organizzare annunci online con supporto AI per l’analisi delle immagini e la moderazione dei contenuti.

## Installazione locale

```bash
git clone https://github.com/dado19994/davide-portfolio.git
cd davide-portfolio
composer install
npm install
cp .env.example .env
php artisan key:generate
npm run dev
php artisan serve
